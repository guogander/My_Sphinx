5. 代码显示¶

5.1. code-block¶

#include<stdio.h>
int main()
{
    printf("%s\n","GYQ");
    return 0;
}

5.2. 显示行号¶

在code-block下使用 :linenos:

1#include<stdio.h>
2int main()
3{
4    printf("%s\n","GYQ");
5    return 0;
6}

5.3. 突出某些行¶

在code-block下使用 :emphasize-lines:

1public class Hello {
2    //在class内部，可以定义若干方法（method）
3    public static void main(String[] args) {// 方法名是main，返回值是void，表示没有任何返回值。
4        // 向屏幕输出文本
5        System.out.println("Hello, World!");//方法代码
6    }
7}

5.4. 引用一个文件¶

使用.. literalinclude:: file 引用文件，在下面使用:lines:可明确引用哪些行

1public class Hello {
2        // 向屏幕输出文本
3        System.out.println("Hello, World!");//方法代码
4    }
5}
