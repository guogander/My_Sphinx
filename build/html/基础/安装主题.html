2. 安装主题¶

以安装sphinx-rtd-theme主题为例： 先安装python库：

pip install sphinx-rtd-theme

然后修改conf.py文件，添加并修改如下内容：

import sphinx_rtd_theme

extensions = [
    "sphinx_rtd_theme",
]

html_theme = "sphinx_rtd_theme"
